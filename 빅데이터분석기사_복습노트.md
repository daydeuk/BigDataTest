### enumerate — 인덱스 + 값 동시에
```python

for i, col in enumerate(cols):
    print(f"{i}번: {col}")
# 0번: age / 1번: income / 2번: score
```

### 조건문 — Python은 and/or 사용 (&&/|| 아님)
```python

    print(score)

if score >= 70 and score < 90:   # and 사용
    print(score)
```

## 2일차 — Pandas 기초 1

### 데이터 불러오기 / 저장
```python
import pandas as pd

df = pd.read_csv('data/train.csv')

# 2유형 답안 제출 시 index=False 필수
# 빠뜨리면 행 번호 컬럼이 생겨 감점
df.to_csv('result.csv', index=False)
```

### 핵심 함수 6개

| 함수 | 용도 |
|------|------|
| `df.head(n)` | 위에서 n행 (기본 5) |
| `df.tail(n)` | 아래에서 n행 |
| `df.shape` | (행 수, 열 수) |
| `df.dtypes` | 전체 컬럼 자료형 |
| `df.info()` | 결측치 + 자료형 요약 |
| `df.describe()` | 수치형 기초 통계 |

```python
df.shape[0]          # 행 수
df.shape[1]          # 열 수
df['age'].dtype      # 특정 컬럼 자료형 (단수)
df.describe(include='all')  # 문자열 컬럼 포함
```

### DataFrame vs Series

| | 설명 | 예시 |
|--|------|------|
| **DataFrame** | 표 전체 (행 + 열) | `df` |
| **Series** | 컬럼 하나 (1차원) | `df['age']` |

```python
type(df)         # pandas.core.frame.DataFrame
type(df['age'])  # pandas.core.series.Series
```

### 결측치 확인

```python
df.isnull().sum()               # 전체 컬럼 결측치 개수
result = df.isnull().sum()
print(result[result > 0])       # 결측치 있는 컬럼만 출력
```

### for문 안에서 dtype 접근

```python
# da는 컬럼명(문자열)이라 da.dtype은 불가
for da in df:
    print(df[da].dtype)   # ✅ df[da]로 Series로 꺼낸 뒤 .dtype
```

### 시험 시작 루틴
```python
df = pd.read_csv('data/train.csv')
df.head()
df.info()             # 결측치 + 자료형 확인
df.describe()         # 기초 통계 확인
df.isnull().sum()     # 결측치 개수 확인
```
