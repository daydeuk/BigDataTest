## 4일차 — 결측치 처리 + 중복 제거

### 결측치 확인
```python
df.isnull().sum().sum()     # 전체 결측치 개수
s = df.isnull().sum()
print(s[s > 0])             # 결측치 있는 컬럼만
```

### 결측치 제거 — dropna()
```python
df = df.dropna()                        # 결측 행 전부 제거
df = df.dropna(subset=['region'])       # 특정 컬럼 기준
df = df.dropna(subset=['age', 'score']) # 둘 중 하나라도 결측이면 제거
```

> 반드시 `df =` 로 재할당. 안 하면 원본이 안 바뀜.

### 결측치 대체 — fillna()
```python
df['col'] = df['col'].fillna(df['col'].mean())    # 평균
df['col'] = df['col'].fillna(df['col'].median())  # 중앙값
df['col'] = df['col'].fillna(df['col'].mode()[0]) # 최빈값 (범주형)
```

### 중복 확인 / 제거
```python
df.duplicated().sum()                          # 중복 행 개수
df.duplicated(subset=['date', 'id']).sum()     # 컬럼 기준

df = df.drop_duplicates()                      # 전체 컬럼 기준 중복 제거
df = df.drop_duplicates(subset=['date', 'id']) # 컬럼 기준
```

### 시험 빈출 패턴
```python
# 중복 제거 → 결측 제거 → 집계 → 정렬 → N번째 출력
df = df.drop_duplicates()
df = df.dropna(subset=['region'])
total = df.groupby('region')['sales'].sum()
result = total[total >= total.mean()].sort_values(ascending=False)
print(result.index[0])
```

> `groupby` 결과는 Series — 인덱스가 그룹 키, 값이 집계 결과.
> `.sort_values(ascending=False).index[0]` 으로 첫 번째 그룹명 출력.

## 3일차 — Pandas 기초 2

### .loc vs .iloc

| | `.loc` | `.iloc` |
|--|--------|---------|
| 기준 | 라벨(이름) | 위치(숫자) |
| 슬라이싱 끝 | **포함** | **미포함** |
| 사용 예 | `df.loc[0, 'name']` | `df.iloc[0, 1]` |

```python
df.loc[0, 'name']               # 0번 행의 name 값
df.loc[0:2, ['name', 'score']]  # 0~2행, 여러 컬럼 (끝 포함)
df.iloc[-1]                     # 마지막 행 (끝 미포함)
df.iloc[0:2, 0:3]               # 0~1행, 0~2열
```

> 마지막 행은 `iloc[-1]`. `loc[-1]`은 라벨 -1이 없으면 에러.

### 조건 필터링

```python
df[df['age'] >= 30]                             # 단일 조건
df[(df['gender'] == 'M') & (df['score'] >= 70)] # AND
df[(df['dept'] == '개발') | (df['age'] <= 25)]  # OR
```

> 조건 여러 개는 각각 `()`로 묶고 `&` / `|` 사용. `and`/`or` 쓰면 에러.

### 컬럼 선택 + 필터링 한 번에

```python
df[['name', 'age']]                          # 컬럼 선택 → DataFrame
df.loc[df['age'] >= 30, ['name', 'age']]     # 조건 + 컬럼 선택 한 줄에
```

> `df[조건][['컬럼']]` 과 `df.loc[조건, ['컬럼']]` 은 동일한 결과. 후자가 더 간결.
