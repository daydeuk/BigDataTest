# Agent.md — 학습 진행 방식 가이드

빅데이터분석기사 실기 학습 시 매 일차마다 동일한 방식으로 진행한다.

---

## 학습 순서

### 1단계 : 개념 설명
- 해당 일차의 핵심 함수/개념을 설명한다
- 각 개념마다 **기본 예제 코드**와 **시험에서 자주 쓰는 패턴**을 함께 제시한다
- Colab에서 바로 실행할 수 있는 형태로 제공한다
- 질문이 나오면 즉시 답변하고 넘어간다

### 2단계 : 연습문제 출제
- 해당 일차에서 배운 내용 기반으로 **5문제** 출제한다
- 문제는 Colab에서 바로 실행 가능한 데이터 세팅 코드를 먼저 제공한다
- 난이도는 기본 → 응용 순서로 구성한다

### 3단계 : 풀이 평가
- 제출된 풀이를 문제별로 ✅ / ❌ / △ 로 평가한다
- 오답은 **이유를 설명**하고 **모범답안**을 제시한다
- △ (부분 점수)는 맞은 부분과 개선할 부분을 구분해서 설명한다
- 추가로 알아두면 좋은 내용(더 나은 방식 등)도 함께 안내한다

### 4단계 : 문서 업데이트
연습문제 완료 후 아래 3개 파일을 업데이트한다.

| 파일 | 업데이트 내용 |
|------|-------------|
| `practice/dayN.md` | 해당 일차 연습문제 + 최종 풀이 저장 |
| `빅데이터분석기사_복습노트.md` | 해당 일차에서 배운 핵심 내용 추가 |
| `빅데이터분석기사_실기_공부계획.md` | 해당 일차 완료 ✅ 표시 |

---

## practice/dayN.md 작성 형식

```markdown
# N일차 연습문제 — [주제]

## 데이터 세팅
(공통으로 사용할 코드)

---

## Q1. [문제 설명]

### 내 풀이 ✅ or ❌
(코드)

### 모범답안 (오답인 경우)
(코드)

> 피드백 메모
```

---

## 복습노트 작성 원칙
- 함수 설명보다 **코드 중심**으로 정리
- 시험에서 실제로 쓰는 패턴 위주
- 주의사항/함정은 `>` 인용 블록으로 강조
