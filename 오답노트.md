## Day 2 — Pandas 기초 1

### `for da in df`에서 dtype 접근
- **실수**: `da.dtype` 으로 접근
- **이유**: `for da in df`에서 `da`는 컬럼명(문자열)이라 `.dtype` 없음
- **정답**: `df[da].dtype` — Series로 꺼낸 뒤 접근
```python
for da in df:
    print(df[da].dtype)  # ✅
```

## Day 4 — 결측치 처리 + 중복 제거

### round() 누락
- **실수**: 소수점 출력 조건에서 `round()` 안 씀
- **정답**: 수치 결과 출력 시 항상 `round(값, 3)` 습관화
```python
print(round(df['sales'].mean(), 3))  # ✅
```

### 재할당 누락 — `df =` 빠뜨리기
- **실수**: `d8.dropna(subset=['region'])` — 결과 저장 안 됨
- **이유**: pandas 대부분의 메서드는 원본을 바꾸지 않고 새 객체를 반환
- **정답**: 반드시 변수에 다시 저장
```python
df = df.dropna(subset=['region'])   # ✅
df = df.drop_duplicates()           # ✅
df = df.fillna(0)                   # ✅
```

### 중복 제거를 iloc 슬라이싱으로 시도
- **실수**: `df.iloc[0:df.shape[0], ...]`으로 중복 제거하려 함
- **이유**: iloc는 위치 기반 슬라이싱 — 중복 제거 기능 없음
- **정답**: `df.drop_duplicates()` 사용
```python
df = df.drop_duplicates()                       # 전체 컬럼 기준
df = df.drop_duplicates(subset=['date', 'id'])  # 컬럼 기준
```
