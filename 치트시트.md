## 3. transform vs groupby().agg() — 가장 자주 틀리는 패턴

| | 반환 | 언제 |
|---|---|---|
| `groupby().mean()` | 그룹 수만큼 (N행) | 그룹별 결과만 필요할 때 |
| `groupby().transform('mean')` | 원본과 같은 행 수 | 행별 비교할 때 |

```python
# ❌ 크기 불일치 — 비교 불가
df[df['score'] >= df.groupby('dept')['score'].mean()]

# ✅ transform으로 각 행에 그룹값 붙이기
df['dept_mean'] = df.groupby('dept')['score'].transform('mean')
df[df['score'] >= df['dept_mean']]
```

## 6. 2유형 전처리 패턴 (Day10~)

### 전체 흐름

```
1. 데이터 로드          pd.read_csv()
2. 결측치 처리          fillna / SimpleImputer
3. 인코딩               get_dummies / LabelEncoder  ← split 전
4. X, y 분리            X = df.drop('target') / y = df['target']
5. train_test_split     test_size, random_state, stratify(분류만)
6. 스케일링             fit_transform(train) / transform(test)  ← split 후
7. 모델 학습            model.fit(X_train, y_train)
8. 예측                 model.predict(X_val)
9. 평가                 accuracy/f1/roc_auc (분류) | RMSE/MAE (회귀)
10. CSV 제출            result.to_csv('result.csv', index=False)
```

### 스케일링 — split 필요 여부 구분

| | split 필요 | 적용 대상 | 이유 |
|---|---|---|---|
| 문제에서 train 기준 fit 요구 | O | train/test 분리 후 | 데이터 누수 방지 |
| 문제에서 그냥 변환만 요구 | X | 전체 df에 바로 적용 | split 언급 없으면 전체 |

```python
# split 필요한 경우
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled  = scaler.transform(X_test)   # fit_transform 금지!

# split 불필요한 경우
scaler = MinMaxScaler()
scaled = scaler.fit_transform(df[['col1', 'col2']])
scaled_df = pd.DataFrame(scaled, columns=['col1', 'col2'])
```

> **핵심**: 문제에 split 언급 있으면 split 후, 없으면 전체 df에 바로 적용

## 7. 분류 모델 평가지표 (Day11~)

### predict vs predict_proba 구분 ★★★

| 메서드 | 반환 | 쓰는 지표 |
|---|---|---|
| `model.predict(X_test)` | 0 또는 1 | accuracy, f1 |
| `model.predict_proba(X_test)[:, 1]` | 0~1 확률값 | roc_auc **만** |

```python
# accuracy, f1 — predict() 사용
pred = model.predict(X_test)
print(round(accuracy_score(y_test, pred), 4))
print(round(f1_score(y_test, pred, average='macro'), 4))

# roc_auc — predict_proba()[:, 1] 사용 (반드시 [:, 1] 붙임)
pred_proba = model.predict_proba(X_test)[:, 1]
print(round(roc_auc_score(y_test, pred_proba), 4))
```

> `[:, 1]` = 모든 행의 1번 열(양성 클래스 확률)만 추출. roc_auc 쓸 때 **항상 세트**로 외울 것.

## 8. 3유형 — 핵심 패턴 5개 ★★★

### p-value 해석 (무조건 암기)
```
p-value < 0.05  → 유의한 차이/관계 있음
p-value >= 0.05 → 유의한 차이/관계 없음
```

### 공통 출력 패턴
```python
from scipy import stats
# 모든 검정 결과
print(round(stat, 3))
print(round(p, 3))
```

---

### 1. 일표본 t검정 — "평균이 기준값과 다른가?"
```python
stat, p = stats.ttest_1samp(df['score'], 70)
```

---

### 2. 대응표본 t검정 — "전후 차이가 있는가?"
```python
stat, p = stats.ttest_rel(df['before'], df['after'])
```

---

### 3. 독립표본 t검정 (Welch) — "두 그룹 평균이 다른가?"
```python
g_a = df[df['group'] == 'A']['score']
g_b = df[df['group'] == 'B']['score']
stat, p = stats.ttest_ind(g_a, g_b, equal_var=False)  # Welch
# equal_var=True → 합동분산 t검정
```

---

### 4. ANOVA — "3개+ 그룹 평균이 다른가?"
```python
g_a = df[df['group'] == 'A']['score']
g_b = df[df['group'] == 'B']['score']
g_c = df[df['group'] == 'C']['score']
stat, p = stats.f_oneway(g_a, g_b, g_c)
```

---

### 5. 카이제곱 — "두 범주형 변수가 관련 있는가?"
```python
ct = pd.crosstab(df['group'], df['passed'])
chi2, p, dof, expected = stats.chi2_contingency(ct)
print(round(chi2, 3))
print(round(p, 3))
print(dof)
# print(round(expected.min(), 3))  # 기대빈도 최솟값 요구 시
```
> 반환 순서: chi2 → p → dof → expected

---

### + F검정 (분산 비교)
```python
var1, var2 = g_a.var(), g_b.var()
F = var1/var2 if var1 >= var2 else var2/var1
df1, df2 = len(g_a)-1, len(g_b)-1
p_f = 2 * min(stats.f.cdf(F, df1, df2), stats.f.sf(F, df1, df2))
print(round(F, 3))
print(round(p_f, 3))
```
