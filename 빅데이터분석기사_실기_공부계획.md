## 3유형 핵심 템플릿

코드가 짧고 패턴이 단순해서 하루만 투자해도 최소 10점 확보 가능.

```python
from scipy import stats

# t-test (두 집단 평균 비교)
result = stats.ttest_ind(group1, group2)
print(result.pvalue)  # < 0.05 이면 귀무가설 기각

# 카이제곱 검정 (범주형 독립성 검정)
result = stats.chi2_contingency(table)
print(result[1])  # pvalue

# ANOVA (세 집단 이상 평균 비교)
result = stats.f_oneway(group1, group2, group3)
print(result.pvalue)
```

**해석 원칙**: p-value < 0.05 → 귀무가설 기각 (유의미한 차이 있음)
